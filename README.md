# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/pavitra-G16/FutureVerse.git
```

Go to project folder

```bash
cd FutureVerse
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build project

```bash
npm run build
```
